PDF 기울기 보정

스캔한 PDF 파일을 선택해서 페이지별 기울기를 자동으로 보정하는 Windows용 데스크톱 앱입니다. 브라우저에 PDF를 업로드하지 않고, 내려받은 앱을 PC에서 실행합니다.

Windows 앱 다운로드 GitHub에서 보기

사용 방법

  1. ZIP 파일을 내려받아 원하는 폴더에 압축을 풉니다.
  2. requirements.txt의 Python 패키지를 설치합니다.
  3. run-gui.bat 파일을 더블클릭합니다.
  4. PDF 파일을 선택하고 저장 위치를 확인한 뒤 보정을 시작합니다.

포함 기능

  • PDF 파일 선택 창
  • 자동 기울기 보정
  • DPI 및 수동 보정 각도 입력
  • 완료 후 저장 경로와 페이지별 보정 각도 표시
